# Studio de l'Imam

## Description du projet

Un studio tout confort équipé d'une salle de bain privative a été créé dans la
mosquée pour l'hébergement ponctuel d'invités religieux. Ce lieu d'accueil
permet de recevoir dignement les Imams et personnalités religieuses lors de
leurs séjours à Saint-Imier.

## Réalisation accomplie

Le studio a été utilisé avec succès, notamment pour héberger l'Imam qui est venu
résider dans nos locaux pendant tout le mois de Ramadan 2024. Cette réalisation
a grandement facilité l'organisation d'événements religieux et l'accueil de nos
invités.

## Dépassement budgétaire justifié

Le budget initial de **CHF 12'000** a été légèrement dépassé pour atteindre
**CHF 14'500**. Ce dépassement s'explique par l'ajout d'éléments de confort
supplémentaires :

- **Salle de bain privative complète** avec douche moderne
- **Mobilier de qualité** adapté aux séjours prolongés
- **Finitions soignées** pour un accueil digne

Ces améliorations, bien que représentant un surcoût de CHF 2'500, ont
considérablement amélioré l'expérience d'hébergement et renforcé la réputation
d'accueil de notre mosquée.

## Impact positif

Ce studio d'accueil :

- Facilite l'organisation d'événements religieux
- Renforce les liens avec la communauté islamique élargie
- Témoigne du sérieux et de l'hospitalité de l'ACMSI
- Constitue un investissement durable pour les années à venir
